# AIナレッジチャット

ChatGPT風のインターフェースでナレッジベースとチャットできるシステムです。

## 機能

- 💬 チャットインターフェース
- 📚 ナレッジカード表示
- 🔍 チャット履歴検索
- 💾 ローカルストレージでの永続化
- ✏️ チャット名前変更
- 🗑️ チャット削除

## 技術スタック

- Next.js 15.3.5
- React 18
- TypeScript
- Tailwind CSS

## 開発環境での実行

```bash
npm install
npm run dev
```

http://localhost:3000 でアクセス可能です。

## 使用方法

1. 「新しいチャット」ボタンでチャットを作成
2. 質問を入力してAIと対話
3. 関連するナレッジカードが表示される
4. サイドバーで過去のチャットを管理・検索
