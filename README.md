# ナレッジチャットシステム (shibu-rain)

## 概要

Obsidian Vault内のナレッジを活用した、ローカルLLM（Qwen2.5 7B）によるチャットシステムです。業界知識と能力開発テーマに関する質問に対して、関連するナレッジファイルを検索し、AIが回答を生成します。

## 🚀 最新機能

### ✅ Ollama統合完了
- **ローカルLLM**: Qwen2.5 7Bモデルを使用
- **日本語対応**: 完全な日本語での質問・回答
- **リアルタイム検索**: ナレッジベースからの関連情報抽出
- **スコアリング**: 関連性に基づく検索結果のランキング

### 🔧 システム構成
- **バックエンド**: FastAPI + Python
- **フロントエンド**: HTML/JavaScript
- **LLM**: Ollama + Qwen2.5 7B
- **ナレッジベース**: Markdownファイル（動的スキャン）

## 📁 ディレクトリ構成

```
shibu-rain/
├── backend/           # FastAPIバックエンド
│   └── main.py       # メインAPIサーバー
├── FRONTEND/         # フロントエンド
│   └── index.html    # チャットインターフェース
├── KNOWLEDGE/        # ナレッジベース
│   ├── 業界別/       # 業界別ナレッジ
│   └── 能力開発テーマ/ # テーマ別ナレッジ
├── INBOX/            # 入力メモ
└── PROCESSING/       # 処理中ファイル
```

## 🛠️ セットアップ

### 1. 依存関係のインストール

```bash
# Python依存関係
python3 -m pip install fastapi uvicorn requests

# Ollamaのインストール（macOS）
brew install ollama

# Qwen2.5 7Bモデルのダウンロード
ollama pull qwen2.5:7b
```

### 2. サーバーの起動

```bash
# バックエンドAPIの起動
cd backend
python3 main.py
```

### 3. ブラウザでのアクセス

```
http://localhost:8000/static/index.html
```

## 💡 使用方法

### チャット機能
1. ブラウザでチャットインターフェースにアクセス
2. 業界や能力開発に関する質問を入力
3. AIが関連ナレッジを検索して回答を生成

### 質問例
- 「建設業界での女性活躍について教えてください」
- 「組織風土改革の成功事例を教えてください」
- 「現場職人の技能伝承について知りたいです」

## 🔍 機能詳細

### ナレッジ検索
- **動的スキャン**: 新規ファイルの自動認識
- **関連性スコアリング**: キーワードマッチングによる関連度計算
- **カテゴリ分類**: 業界別・テーマ別の整理

### LLM応答生成
- **コンテキスト活用**: 関連ナレッジファイルの内容を活用
- **日本語最適化**: 日本語での自然な回答生成
- **実用的アドバイス**: 具体的で実践的な回答
